Capstone Project

For my final project at NSCC, my class is developping a secure and scalable network infrastructure to support a carpool service for the campus. The goal was to create a reliable platform that allows students to coordinate shared rides efficiently and securely.

I am part of the network and security team, together we designed a network and simulated it using GNS3, incorporating best practices in segmentation, load balancing, security, and high availability. Key components include internal and edge firewalls, NGINX web server, DNS, a management VLAN, and a cluster of worker nodes for scalability.

  • GNS3 Network Simulation
  • NGINX Web Server
  • Firewall & VLAN Configuration
  • Kubernetes
  • Linux Administration

Designing the network architecture taught me about scalable infrastructure, VLAN segmentation, and securing services through internal and edge firewalls. It pushed me closer toward my goal of working in network security or systems administration.
